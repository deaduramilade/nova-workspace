# FAQ.md

## Frequently Asked Questions

### General Questions

**What is nova-workspace?**  
nova-workspace is a professional open-source workspace platform designed for both enterprise and consumer users, emphasizing efficiency, reliability, and extensibility.

**Who is this project for?**  
Developers, teams, and organizations seeking a high-quality, maintainable workspace solution with strong documentation and agent-assisted development workflows.

**Is nova-workspace production-ready?**  
The project is under active development. Check ROADMAP.md and CHANGELOG.md for current maturity level.

### Installation and Setup

**What are the minimum requirements?**  
See INSTALL.md for detailed prerequisites and setup instructions.

**Why use Docker for installation?**  
Docker provides consistent environments, easier dependency management, and better isolation for production-like testing.

**I encountered an error during installation. What should I do?**  
Check the error message, ensure all prerequisites are met, and consult the troubleshooting section in INSTALL.md. If unresolved, open an issue with reproduction steps.

### Development and Agent Usage

**How do I interact with the AI agent?**  
Follow the guidelines in AGENT.md. The agent reads core documentation files at the start of every session and maintains LOG.md for continuity.

**Why are there so many .md documentation files?**  
They enable structured knowledge management, agent self-learning, consistency, and transparency for contributors and enterprise users.

**How can I minimize credit consumption when using the agent?**  
Be specific in requests, reference existing patterns in PATTERNS.md and LOG.md, and allow the agent to leverage prior work.
