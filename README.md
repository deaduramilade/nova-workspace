# Nova Workspace

**AI-native collaborative browser workspace** — Docker + WebRTC + Native AI Agents (Ollama-powered)

An open-source, self-hostable platform where humans and AI agents collaborate in real-time inside isolated Docker environments.

**Apache-2.0 License** • **Phase 1** (Foundation & Docker Setup)

---

## ✨ Key Features

- Real-time collaboration via WebRTC
- Native AI agents as first-class participants
- Full offline-first support with 7-day encrypted retention
- Strong privacy, security & global compliance (GDPR, SOC 2, etc.)
- Optimized for low-memory and low-data environments
- Easy one-command deployment
